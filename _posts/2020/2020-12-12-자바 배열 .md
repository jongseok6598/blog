배열이란 데이터를 묶어주는 것이다     

배열선언 방법
```
int[] arr = {10, 30, 20, 50, 3, 60, -3};
//한번에 다 선언하기
int[] num = new int[3];//크기가 3인배열생성
num[0] = 10;//첫번째 값을 10으로 선언(배열은 0부터 시작)
num[1] = 15;//두번째 값을 15로 선언
num[2] = 13;//마지막 값을 13으로 선언
//따로따로 선언하기
```
