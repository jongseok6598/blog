생성자   
인스턴스를 생성할때 호출되는 처리   

생성자의 두가지 특징   
1. 메서드명이 클래스명과 동일하다
2. 반환값의 선언이 존재하지 않음
```
public class Animal {
    String name;
    public void setName(String name) {//setter메서드
        this.name = name;
    }

    public Animal(String name) {//생성자 선언
        this.setName(name);
    } 

    
}
```
```
public static void main(String[] args) {//생성자가 잘 선언되었는지 확인
    Animal dog = new Animal("cat"); 
    System.out.println(dog.name);
}
```
