생성자   
객체변수에 값을 무조건 설정해야만 생성될수 있게 만드는것   

생성자사용 방법
```
public class Animal {
    String name;//클레스 내에있는 name변수
    public void setName(String name) {//name을 선언하는 메소드
        this.name = name;
    }

    public Animal(String name) {//생성자 메소드
        this.setName(name);
    } 

    
}
```
```
public static void main(String[] args) {
    Animal dog = new Animal("cat");//dog.name을 cat이라고 설정함  
    System.out.println(dog.name);//dog.name을 cat라고 설정해서 cat 출력
}
```
