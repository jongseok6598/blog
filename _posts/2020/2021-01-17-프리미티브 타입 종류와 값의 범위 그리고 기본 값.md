타입
타입이란 프로그램안에서 취급하는 데이터 형식을 정의한 것이다.

프리미티브 타입   
프리미티브 타입은 기본적인 데이터 타입이다
```
short shortNum;//정수형(16비트)
int intNum;//정수형(32비트)
char character;//문자형
float floatNum;//실수형(32비트)
double doubleNum;//실수형(64비트)
String str;//문자열
```
